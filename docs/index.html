Identify Hardening Soil parameters from lab tests — in one second

Upload a drained triaxial test and an oedometer test. The XGBoost surrogate returns the full HS parameter set, ready for PLAXIS.

XGBoost · 34 011 PLAXIS simulations 12 HS parameters ×19 500 faster than manual calibration ε1 ≤ 0.25 · model limit
01

Upload files

Triaxial q–ε1, εvol–ε1, stress path and oedometer with loading–unloading cycle

02

Interpolation

Done server-side — Ridge polynomial regression to 250 uniform strain points (triaxial) and 750 points (oedometer)

03

ML prediction

3 000-feature vector → XGBoost → 12 HS parameters. R² ≥ 0.98 on test set

04

Verify in PLAXIS

Run SoilTest script, export curves, upload back to overlay on lab data
